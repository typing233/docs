# 欢迎使用 Scenext API

Scenext 是一个AI视频生成平台，能够根据提供的问题和参考答案内容自动生成高质量的讲解视频。

## 主要功能

- **AI视频生成**: 基于文本和图片内容自动生成视频
- **多质量选择**: 支持不同质量级别的视频输出
- **异步处理**: 视频生成任务异步处理
- **实时状态**: 实时查询视频生成状态和进度
- **回调通知**: 任务完成时自动通知您的系统

## API特点

- **简单易用**: RESTful API设计，易于集成
- **安全可靠**: 基于Token认证和HMAC签名验证

## 快速开始

1. 访问 [scenext.cn](https://scenext.cn) 注册账户
2. 在个人中心创建API密钥
3. 开始调用API生成视频

准备好开始了吗？查看我们的[快速开始指南](/cn/quickstart)。
