## API 概述

Scenext API 提供以下核心功能：

- **视频生成** - 基于题目内容自动生成讲解视频
- **任务管理** - 异步任务状态查询
- **回调通知** - 任务完成时的安全回调
- **积分计费** - 按视频分镜数量计费

更多详细信息请查看相应的文档页面。
