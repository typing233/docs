# Welcome to Scenext API

Scenext is an AI video generation platform that automatically creates high-quality explanatory videos based on your provided question and reference answer content.

## Key Features

- **AI Video Generation**: Automatically generate videos based on text and image content
- **Multiple Quality Options**: Support different quality levels for video output
- **Asynchronous Processing**: Video generation tasks are processed asynchronously
- **Real-time Status**: Real-time query of video generation status and progress
- **Callback Notifications**: Automatic notification to your system when tasks complete

## API Features

- **Easy to Use**: RESTful API design, easy to integrate
- **Secure & Reliable**: Based on Token authentication and HMAC signature verification

## Quick Start

1. Visit [scenext.cn](https://scenext.cn) to register an account
2. Create API keys in your personal center
3. Start calling the API to generate videos
